- ContactSystem -


ContactSystem (이)라는 class를 통해 연락처에 접근

AContact (이)라는 class는 연락쳐 정보를 저장



class AContact{

    name:String // 이름 (성 + 이름)

    phoneNum:String // 전화번호 한 연락쳐에 저장된 번호중 첫번쨰로 등록된 번호

}                                                                



name : 성 + 이름 ( familyName + givenName )

phoneNum : 전화번호 ( phoneNumbers[0] )


enum type {

    case phoneNumber , name

}                 


type (이)라는 enum은 [AContact]를 가져올떄 어떤 형식을 통해 가져올지

phoneNumber : 전화번호으로 가져오기

name : 이름으로 가져오기


class ACList{

    var acs:[AContact]

}    

 

내가 전화번호 리스트 만들려고 만든거


class ContactSystem{ func getContact(getType:type , settedValue:String) -> [AContact]

func saveContact(firstName:String , secondName:String , phoneNum:String)

}    

 

ContactSystem : 주요 class

getContact : 연락쳐들 가져오기

getType : 가져올 방식

settedValue : 연락쳐에 어떤 데이터가 있어야 하는지


saveContact : 연락쳐 저장

firstName : 성

secondName : 이름

phoneNum : 전화번호 ( 여려게 X )



예제들


saveContact(firstName: "멍" , secondName: "청이" , phoneNum: "+821000000000")


+821000000000이라는 전화번호를 가진 멍청이라는 이름을 가진 사람의 연락쳐를 iPhone에 저장


getContact(getType:.phoneNumber , settedValue: "+821000000000")


"+821000000000"이란 String이 전화번호에 포함되있는 [AContact]를 반환


getContact(getType:.name , settedValue: "청이")


"청이"이란 String이 이름에 포함되있는 [AContact]를 반환
